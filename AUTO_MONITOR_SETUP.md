## ⏳ 需要你操作（5 分钟）

### 启用定时检查

**方法 1: crontab（推荐）**
```bash
# 1. 编辑 crontab
crontab -e

# 2. 添加这行
0 */4 * * * ~/.openclaw/scripts/capability-watch.sh check >> ~/.openclaw/logs/capability-watch.log 2>&1

# 3. 保存退出
```

**方法 2: 手动测试（先验证）**
```bash
# 手动运行一次
~/.openclaw/scripts/capability-watch.sh check

# 查看日志
cat ~/.openclaw/logs/capability-watch.log
```
