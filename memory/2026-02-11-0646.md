print(f"Using device: {device}")

# 简单 MLP 示例
class SimpleMLP(nn.Module):
    def __init__(self):
        super().__init__()
        self.layers = nn.Sequential(
            nn.Linear(784, 512),
            nn.ReLU(),
            nn.Linear(512, 10)
        )
    
    def forward(self, x):
        return self.layers(x)

#### 4.1 强化学习基础

**核心概念框架：**

```
RL 问题结构：
┌─────────────────────────────────────┐
│  Agent ←→ Environment               │
│  • State (s) - 当前状态              │
│  • Action (a) - 动作选择             │
│  • Reward (r) - 奖励信号             │
│  • Policy (π) - 策略函数             │
│  • Value (V) - 价值函数              │
└─────────────────────────────────────┘
```

**经典算法：**
1. **DQN** (Deep Q-Network) - 值函数方法
