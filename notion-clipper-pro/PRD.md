## 1. 产品概述

### 1.1 产品定位
Notion Clipper Pro 是一款浏览器插件，帮助用户将网页内容智能裁剪并保存到 Notion 笔记中。

### 1.2 目标用户
- Notion 重度用户
- 知识管理爱好者
- 研究人员/学生
- 内容创作者

### 1.3 核心价值
- **高效**: 一键保存，无需手动复制粘贴
- **智能**: 自动识别正文，过滤无关内容
- **规范**: Markdown 格式，完美兼容 Notion
- **隐私**: 纯前端实现，数据不上传

## 2. 功能需求

### 2.1 核心功能

#### F1: 内容提取

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| F1.1 | 用户手动选中内容保存 | P0 |
| F1.2 | 自动识别页面正文 | P0 |
| F1.3 | 智能过滤广告/导航 | P0 |
| F1.4 | 提取元数据（标题/作者/时间） | P0 |

**验收标准**:
- 选中文字后右键可见"保存到 Notion"选项
- 点击插件图标可见"保存整页"选项
- 自动识别准确率 > 80%
- 元数据提取完整

#### F2: 格式转换

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| F2.1 | HTML 转 Markdown | P0 |
| F2.2 | 保留基本格式（粗体/斜体/链接） | P0 |
| F2.3 | 添加元数据头部 | P0 |
| F2.4 | 清理多余空白 | P1 |

**验收标准**:
- 转换后的 Markdown 可在 Notion 中正常显示
- 格式保留完整
- 元数据包含来源 URL、保存时间

#### F3: 用户交互

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| F3.1 | 右键菜单 | P0 |
| F3.2 | 工具栏弹出界面 | P0 |
| F3.3 | 快捷键支持 | P0 |
| F3.4 | 成功/失败提示 | P0 |

**验收标准**:
- 右键菜单响应迅速
- 弹出界面美观易用
- 快捷键可配置
- 提示信息清晰

#### F4: 剪贴板集成

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| F4.1 | 自动复制到剪贴板 | P0 |
| F4.2 | 复制成功提示 | P0 |
| F4.3 | 失败降级方案 | P1 |

**验收标准**:
- 复制成功率 > 95%
- 用户可立即粘贴到 Notion
- 失败时有明确提示

## 4. 用户体验

### 4.1 用户流程

```
保存整页:
1. 用户点击插件图标
2. 点击"保存整页"按钮
3. 系统提取内容并转换
4. 复制到剪贴板
5. 显示成功提示
6. 用户到 Notion 粘贴

保存选中:
1. 用户选中网页文字
2. 右键点击
3. 选择"保存到 Notion"
4. 系统处理选中内容
5. 复制到剪贴板
6. 显示成功提示
```

### 4.2 界面设计

#### 弹出界面
```

│  [✂️ 保存选中内容]      │
├─────────────────────────┤
```
