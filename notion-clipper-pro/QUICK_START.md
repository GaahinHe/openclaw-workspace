## ⚡ 超快安装（3 步）

### 步骤 1: Edge 已打开 ✅

我已帮你打开 Edge 浏览器到扩展页面：
```
edge://extensions/
```

### 步骤 2: 开启开发者模式（10 秒）

在 Edge 扩展页面：
1. 找到右上角 **"开发者模式"** 开关
2. 点击打开（变蓝色）

### 步骤 3: 加载插件（20 秒）

1. 点击 **"加载已解压的扩展程序"** 按钮
2. 在弹出的窗口中，选择：
   ```
   /Users/hans/.openclaw/workspace/notion-clipper-pro
   ```
3. 点击 **"选择文件夹"**

## ❓ 遇到问题？

### 问题：找不到"加载已解压的扩展程序"

**解决**：先开启右上角的"开发者模式"

### 问题：图标不显示

**解决**：刷新页面或重启 Edge

### 问题：复制失败

**解决**：确保不是 edge:// 开头的特殊页面
