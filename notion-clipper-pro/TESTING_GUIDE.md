## 🚀 方法 1: 自动安装脚本（推荐）

### 步骤 1: 运行安装脚本

```bash

cd ~/.openclaw/workspace/notion-clipper-pro

bash install-to-edge.sh
```

### 步骤 2: 跟随提示操作

2. ✅ 验证插件文件
3. ✅ 打开 Edge 扩展页面
4. 📝 提示你手动开启开发者模式并加载

### 步骤 3: 手动加载（30 秒）

Edge 会打开到 `edge://extensions/`

1. **开启开发者模式**
   - 右上角找到"开发者模式"开关

2. **加载插件**
   - 点击"加载已解压的扩展程序"
   - 选择文件夹：`~/.openclaw/workspace/notion-clipper-pro/`
   - 点击"选择文件夹"

3. **验证成功**
   - 看到 Notion Clipper Pro 出现在列表中
   - 工具栏出现 📋 图标
   - ✅ 加载成功！

## 🐛 故障排查

### 问题 1: 找不到"加载已解压的扩展程序"

**原因**: 开发者模式未开启

**解决**:
```
1. 找到右上角"开发者模式"开关
2. 打开它（变蓝色）
3. 按钮应该会出现
```

---

### 问题 2: 提示"清单文件缺失"

**原因**: manifest.json 不在文件夹根目录

**解决**:
```
1. 确认选择的是 notion-clipper-pro 文件夹
2. 不是子文件夹（如 popup/ 或 icons/）
3. manifest.json 应该在文件夹根目录
```

---

### 问题 3: 图标不显示

**原因**: PNG 图标文件缺失

**解决**:
```bash
# 检查图标文件
ls -la ~/.openclaw/workspace/notion-clipper-pro/icons/

# 应该有：
# icon16.png, icon32.png, icon48.png, icon128.png
```

---

### 问题 4: 点击没反应

**原因**: 内容脚本未注入

**解决**:
```
1. 刷新页面（Cmd+R）
2. 重新点击插件图标
3. 如果还不行，重启 Edge
```

---

### 问题 5: 复制失败

**原因**: 浏览器权限或特殊页面

**解决**:
```
1. 确保不是 edge:// 或 chrome:// 页面
2. 检查剪贴板权限
3. 尝试手动复制（Cmd+C）
4. 查看控制台错误（F12）
```

---

### 问题 6: 快捷键不工作

**原因**: 快捷键冲突或未注册

**解决**:
```
1. edge://extensions/shortcuts
2. 检查快捷键是否被占用
3. 可以修改为其他组合键
```
