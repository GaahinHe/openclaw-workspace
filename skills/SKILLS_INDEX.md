## 使用指南

### 按需调用技能

1. **遇到编码问题** → 调用「编码能力」类技能
2. **需要复习测试** → 调用「学习辅导」类技能
3. **进行深度探讨** → 调用「深度讨论」类技能
4. **管理记忆文件** → 调用「内存管理」类技能

### 技能组合使用

- **学习新概念**：「记忆框架构建」+「Feynman Technique」
- **深度讨论**：「苏格拉底式提问」+「逻辑分析」+「观点碰撞」
- **代码审查**：「代码审查」+「重构建议」+「内存管理」（记录审查结果）
- **定期复习**：「复习测试」+「3-2-1回顾法」+「长期记忆写入」

### 技能文件结构

每个技能文件应包含：
- 技能名称和描述
- 使用场景
- 调用方式
- 示例对话
- 相关技能链接
