# 学习进度追踪技能 (Progress Tracking)

## 描述
记录和追踪学习里程碑，可视化学习进展

## 使用场景
- 记录学习开始/完成时间
- 追踪技能掌握程度
- 设定和检查里程碑
- 生成学习报告

## 进度模型

### 1. 百分比进度
**结构**:
```json
{
  "topic": "机器学习",
  "progress": 35,
  "started": "2026-02-01",
  "last_updated": "2026-03-10",
  "milestones": [
    {"name": "基础概念", "completed": true, "date": "2026-02-10"},
    {"name": "监督学习", "completed": true, "date": "2026-02-20"},
    {"name": "无监督学习", "completed": false, "date": null},
    {"name": "深度学习", "completed": false, "date": null}
  ]
}
```

### 2. 技能树模型
**结构**:
```
机器学习 (35%)
├── 基础理论 (80%) ✅
│   ├── 统计学基础 ✅
│   ├── 线性代数 ✅
│   └── 微积分 ✅
│
├── 监督学习 (60%) 🟡
│   ├── 线性回归 ✅
│   ├── 逻辑回归 ✅
│   ├── 决策树 ✅
│   └── SVM 🟡
│
├── 无监督学习 (20%) 🔴
│   ├── 聚类 🟡

│
└── 深度学习 (10%) 🔴
    ├── 神经网络基础 🟡
```

**状态图例**:
- ✅ 已完成 (掌握 >80%)
- 🟡 进行中 (掌握 30-80%)
- 🔴 未开始 (掌握 <30%)
