# 复习测试技能 (Review Testing)

## 描述
生成测试题目检验掌握程度，主动 recall 强化记忆

## 使用场景
- 学习后检验理解
- 定期复习测试
- 考前自测
- 发现知识盲区

## 测试类型

### 1. 概念题 (Concept Questions)
**目的**: 检验基础概念理解

**题型**:
- 选择题
- 判断题
- 填空题
- 名词解释

**示例**:
```
【选择题】机器学习的核心是？
A. 编写复杂算法
B. 从数据中学习规律
C. 提高计算速度
D. 存储大量数据

答案：B

【判断题】深度学习一定比传统机器学习效果好。
答案：错（取决于数据和场景）

【填空题】监督学习需要______数据。
答案：带标签

【名词解释】什么是过拟合？
答案：模型在训练集上表现好但在测试集上表现差的现象
```

### 2. 应用题 (Application Questions)
**目的**: 检验知识应用能力

**题型**:
- 场景分析
- 代码填空
- 方案设计
- 问题诊断

**示例**:
```
【场景分析】
你有 100 万条用户数据，要预测用户是否会流失。
1. 这是什么类型的问题？
2. 你会选择什么算法？为什么？
3. 如何评估模型效果？

【代码填空】
补全以下代码：
```python
from sklearn.model_selection import ______
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=______)
```

【方案设计】
设计一个垃圾邮件分类系统，包括：
1. 数据收集
2. 特征工程
3. 模型选择
4. 评估方法

【问题诊断】
模型训练准确率 99%，测试准确率 60%，可能是什么原因？
如何解决？
```

### 3. 深度思考题 (Deep Thinking Questions)
**目的**: 检验深度理解和批判性思维

**题型**:
- 对比分析
- 批判性评估
- 第一性原理
- 扩展思考

**示例**:
```
【对比分析】
比较以下算法的异同：
- 逻辑回归 vs 决策树
-  bagging vs boosting
- 聚类 vs 分类

【批判性评估】
"深度学习将取代所有传统机器学习算法"
你同意这个观点吗？为什么？

【第一性原理】
机器学习的本质是什么？
从信息论角度如何理解学习过程？

【扩展思考】
如果数据量无限大，什么算法最可能表现最好？
为什么？
```

## 测试流程

### 1. 准备阶段
```
1. 确定测试范围
   - 章节/主题
   - 难度级别
   - 题型分布

2. 设计测试结构
   - 概念题：40%
   - 应用题：40%
   - 深度题：20%
   - 总题数：10-20 题
   - 预计时间：30-60 分钟
```

### 2. 实施阶段
```
1. 呈现题目
   - 一次一题或分组呈现
   - 给予充分思考时间

2. 收集答案
   - 记录用户回答
   - 不急于纠正

3. 提供反馈
   - 先肯定正确部分
   - 指出错误和原因
   - 给出正确答案
```

### 3. 评估阶段
```
1. 评分
   - 客观题：自动评分
   - 主观题：按要点评分

2. 分析
   - 正确率统计
   - 薄弱点识别
   - 时间分析

3. 建议
   - 复习重点
   - 下一步学习
   - 补充材料
```

## 评分标准

### 客观题
```
选择题：每题 1 分
判断题：每题 1 分
填空题：每空 1 分
```

### 主观题
```
名词解释 (5 分):
- 5 分：完整准确，有例子
- 4 分：准确但不完整
- 3 分：基本正确
- 2 分：部分正确
- 1 分：沾边
- 0 分：错误

应用题 (10 分):
- 9-10 分：完整方案，考虑周全
- 7-8 分：方案可行，有小遗漏
- 5-6 分：基本思路正确
- 3-4 分：部分正确
- 1-2 分：思路不清
- 0 分：错误
```

## 示例对话

**助手**: 我们来测试一下你对机器学习的理解。

**第一题【概念题】**: 
什么是监督学习和无监督学习的区别？
各举一个例子。

**用户**: 监督学习有标签，无监督学习没有标签。
监督学习例子：分类
无监督学习例子：聚类

**助手**: 

## ✅ 正确！

**你的答案**: 完整准确

**补充**:
- **监督学习**: 输入数据包含标签，目标是学习输入到输出的映射
  - 例子：垃圾邮件分类（标签：垃圾/非垃圾）
  
- **无监督学习**: 输入数据无标签，目标是发现数据内在结构
  - 例子：客户分群（自动发现相似客户）

**延伸思考**:
还有半监督学习，介于两者之间，你知道是什么吗？

---

1. 这是什么类型的问题？
2. 你会选择什么算法？为什么？
3. 如何评估模型？

**用户**: [回答]

...

**用时**: 15 分钟

**表现**:
