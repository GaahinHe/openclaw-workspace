# 记忆维护技能 (Memory Maintenance)

## 描述
定期整理和归档旧记忆，保持记忆系统健康和高效

## 使用场景
- 定期清理过期信息
- 归档历史记录
- 合并重复内容
- 优化记忆结构

## 维护周期

### 每日维护 (自动)
**时机**: 每次会话结束前

**任务**:
- [ ] 检查临时信息是否需要升级
- [ ] 记录重要对话到 daily note
- [ ] 更新任务状态

### 每周维护 (建议)
**时机**: 每周一次 (如周日)

**任务**:
- [ ] 回顾本周 daily notes
- [ ] 提取重要信息到 MEMORY.md
- [ ] 清理过期待办
- [ ] 检查记忆一致性

### 每月维护 (必须)
**时机**: 每月最后一天

**任务**:
- [ ] 完整审查 MEMORY.md
- [ ] 归档过期信息
- [ ] 合并重复记录
- [ ] 优化结构
- [ ] 生成月度记忆报告

### 季度维护 (深度)
**时机**: 每季度末

**任务**:
- [ ] 全面审查所有记忆文件
- [ ] 重构记忆结构 (如需要)
- [ ] 删除无用信息
- [ ] 生成季度学习报告

## 维护操作

### 1. 信息归档

**何时归档**:
- 信息过期但仍可能有参考价值
- 项目结束需要保存历史
- 决策被新决策替代

**归档方法**:
```markdown
# 原位置 (MEMORY.md)
### 历史配置 (已归档)
**归档日期**: 2026-03-01
**原因**: 系统升级，配置已变更

原配置内容...

**当前配置**: `MEMORY.md: 最新配置`

---

# 归档位置 (memory/archive/config-2026-02.md)
## 2026 年 2 月配置
[详细历史配置]
```

### 2. 内容合并

**何时合并**:
- 多处记录同一主题
- 分散的信息可以整合
- 重复的待办事项

**合并方法**:
```markdown
# 合并前
### X 配置 (2026-02-01)
[配置内容 A]

### X 配置更新 (2026-02-15)
[配置内容 B]

# 合并后
### X 配置
**最后更新**: 2026-02-15

**变更历史**:
| 日期 | 变更 | 
|------|------|
| 2026-02-01 | 初始配置 |
| 2026-02-15 | 更新 Y 参数 |

**当前配置**:
[最新配置内容]
```

### 3. 过期清理

**何时清理**:
- 待办事项已完成超过 30 天
- 临时信息已无用
- 过时的计划和安排

**清理方法**:
```markdown
# 清理前
### 待办事项
- [x] 完成任务 A (2026-01-15)
- [x] 完成任务 B (2026-01-20)
- [ ] 进行中任务 C

# 清理后
### 待办事项
- [ ] 进行中任务 C

### 已完成 (最近 30 天)
- [x] 任务 C (2026-02-28)

### 历史已完成
详见 `memory/completed-2026-01.md`
```

### 4. 结构优化

**何时优化**:
- 文件过大 (>100KB)
- 结构混乱难以导航
- 新增类别需要重组

**优化方法**:
```markdown
# 优化前
MEMORY.md (500 行，单一文件)

# 优化后
MEMORY.md (50 行，核心摘要)
├── memory/core-principles.md
├── memory/user-preferences.md
├── memory/tech-stack.md
├── memory/decisions.md
└── memory/todos.md
```

## 自动化维护

### 1. 自动归档脚本
```python
def auto_archive_old_items():
    """自动归档 90 天前的项目"""
    cutoff_date = now() - timedelta(days=90)
    
    for item in memory_items():
        if item.date < cutoff_date and not item.archived:
            archive_item(item)
            log(f"Archived: {item.title}")
```

### 2. 重复检测
```python
def detect_duplicates():
    """检测重复内容"""
    items = load_all_items()
    duplicates = []
    
    for i, item1 in enumerate(items):
        for item2 in items[i+1:]:
            similarity = calculate_similarity(item1, item2)
            if similarity > 0.8:
                duplicates.append((item1, item2, similarity))
    
    return duplicates
```

### 3. 一致性检查
```python
def check_consistency():
    """检查信息一致性"""
    issues = []
    
    # 检查日期逻辑
    for decision in decisions():
        if decision.date > decision.implementation_date:
            issues.append(f"日期矛盾：{decision.title}")
    
    # 检查引用有效性
    for ref in all_references():
        if not ref.exists():
            issues.append(f"无效引用：{ref}")
    
    return issues
```
